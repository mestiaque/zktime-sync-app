# ZKTeco Sync Tool

## Features
- Sync attendance logs from multiple K40 devices
- Push logs to Laravel API
- Simple Windows UI with progress log
- Local JSON backup

## Installation
1. Install Python 3.10+
2. Install requirements: `pip install -r requirements.txt`
3. Run `app.py` or build `.exe` using PyInstaller

## Config
- Add API URL
- Add devices (IP + Password)
- Click 'Sync Now' to push logs

## Notes
- Tkinter comes with Python 3 by default
- Device password is required to connect
